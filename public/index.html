The Todo API

This API is for making simple Todo lists. To use it, you will call various API paths.

Bucket Id

Todo list items are stored by buckets, each having an Id. You can generate a bucket id by navigating to /api/bucketId on this site. You'll see something like this. This is your bucket id.

{
  "bucketId" : "8eba54b1-7680-4eab-a6eb-d1abb78e9974"
}

All requests to the API will expect the bucket id as a query string parameter named bucketId.

Creating a Todo Item

With your bucket id, you can go ahead and create Todo items. To do that, do an HTTP POST to /api/item with the following data.

Request

POST /api/item?bucketId={bucket-id}

text=Go+sell+your+pants

This API will return you the item with a little bit more info. You don't have to use it but it's there if you need it.

Response

{
  "id" : "9a26a94a-6825-48a9-9f22-0b4420978656",
  "text" : "Go sell your pants",
  "isComplete" : false
}

Getting a List of Items

Once you have created at least one item, you can now go get it/them. To do that, make an HTTP GET request to /api/items.

Request

GET /api/items?bucketId={bucked-id}

Response

{
  "items" : [
    {
      "id" : "19e236a0-4903-4c8d-9ba3-5cefeb1186ba",
      "text" : "Go sell your pants",
      "isComplete" : false
    },
    {
      "id" : "36d67be3-b851-405d-9699-134d5cf9f57a",
      "text" : "Buy shorts",
      "isComplete" : true
    },
    {
      "id" : "981441a1-05be-4aa0-8190-2fcf4f203f93",
      "text" : "Put on shorts",
      "isComplete" : false
    }]
}

Deleting Items

To delete an item, do a DELETE request to /api/item/{item-id}, where item-id is the id assigned to the item by the API.

Request

DELETE /api/item/{item-id}?bucketId={bucket-id}

Response

204 No Content

Marking Items Complete

Request

POST /api/item/{item-id}/togglestatus?bucketId={bucket-id}

Response

204 No Content
